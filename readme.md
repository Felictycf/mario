## 项目基础简介

开发环境：Microsoft Visual Studio Community 2022 (64 位) - Current 版本 17.0.0

开发语言：C++

使用图形库：[Easy Graphics Engine(EGE)](https://xege.org/)

## 操作方式

左/右方向键：向左/右移动

下方向键：下蹲

X 键：跳跃（在一定范围内，按下时间越长，横向速度越快，跳跃高度越高）

Z 键：加速
